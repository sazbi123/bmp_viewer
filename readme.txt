bmpファイルビューア

1から32bitのBMPファイルが見れます

使用方法：
python3 bmp_viewer.py bmpファイルのパス

これが正常に実行できると
・bmp_header_dump.txt
・bmp_img_data.txt
の2種類が実行ディレクトリに生成されます
デバッグ用なので本来は必要ありません

Tkinterを使ってます
リサイズできません
常に画像ファイルの縦横サイズのウィンドウができます
できれば直します
あと処理が遅いです
多分リストに色情報をすべて逐次的に入れているせいです
1から4bitのBMPファイルはあまりよろしくないであろう処理をしています
直せたら直す予定です
